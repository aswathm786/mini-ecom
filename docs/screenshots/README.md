## How to Take Screenshots

### On Windows
1. Press `Win + Shift + S` to open Snipping Tool
2. Select area or full screen
3. Save as PNG

### On macOS
1. Press `Cmd + Shift + 4` to select area
2. Screenshot saved to Desktop
3. Move to this directory

### On Linux
1. Use `gnome-screenshot` or `scrot`
2. Save as PNG format
3. Move to this directory

## Image Guidelines

- **Format:** PNG (preferred) or JPG
- **Size:** Maximum 2MB per image
- **Resolution:** 1920x1080 or higher
- **Naming:** Use lowercase with underscores (e.g., `admin_dashboard.png`)
- **Optimization:** Compress images before committing (use tools like TinyPNG)

## Adding Screenshots to Documentation

In markdown files, reference screenshots like this:

```markdown
![Admin Dashboard](/docs/screenshots/admin_dashboard.png)
```

## Privacy & Security

**Important:** Before taking screenshots:
- Remove or blur any sensitive information (emails, phone numbers, addresses)
- Remove or blur any real payment information
- Use test data only
- Don't include real customer information
